# Supporter Endorsement Magic Links

## How it works
1. Each supporter has a unique private link
2. When they visit, they see a simple form to write a quote (200 chars max)
3. Quote appears on their card on the homepage
4. They can update their quote anytime using the same link

## Public Endorsement
Anyone can endorse Laksh via the "Endorse Laksh" button on the homepage.
Opens Twitter with pre-filled text.
